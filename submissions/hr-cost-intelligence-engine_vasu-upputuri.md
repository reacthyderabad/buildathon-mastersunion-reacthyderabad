
# HR Cost Intelligence Engine

## Attendee Name
Vasu Upputuri

## Problem Statement Selected
PS #1 – HR Cost Intelligence Engine

## Project Description
An AI-powered application that converts employee meeting activity into project-level HR cost insights. It automatically calculates meeting costs using employee hourly rates, attributes meetings to projects, and provides dashboards for budget tracking and analytics.

## Approach
- Built with React + TypeScript + Vite for the frontend.
- FastAPI + SQLite backend.
- Employee hourly cost mapping.
- Automatic meeting cost calculation.
- Project-wise cost dashboards and analytics.
- Predict vs Actual cost comparison.
- Seeded demo data for realistic testing.

## Tools and Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- FastAPI
- Python
- SQLAlchemy
- SQLite
